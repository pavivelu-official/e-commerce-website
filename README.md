# E-Commerce Website (UG Project)

## Project Overview
This is a placeholder repository for an Undergraduate project on an E-Commerce Website.  
The project is designed to simulate a basic online shopping platform where users can browse products, add them to the cart, and checkout.

- User Registration & Login
- Product Catalog
- Shopping Cart
- Order Management
- Admin Dashboard
